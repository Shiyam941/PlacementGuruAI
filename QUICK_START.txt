## How to Use

### 1️⃣ Sign Up
```bash
curl -X POST http://localhost:8080/api/auth/signup \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Your Name",
    "email": "your@email.com",
    "password": "Password@123",
    "skillLevel": "BEGINNER"
  }'
```

### 2️⃣ Login (Get Token)
```bash
curl -X POST http://localhost:8080/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{
    "email": "your@email.com",
    "password": "Password@123"
  }'

# Copy the token from response
```

### 3️⃣ Use APIs with Token
```bash
curl -X GET http://localhost:8080/api/chatbot/history \
  -H "Authorization: Bearer YOUR_TOKEN_HERE"
```

---

## Public Endpoints (No login needed)
- `GET /api/auth/test` - Health check
- `GET /api/coding/problems` - View all problems
- `GET /api/coding/problems/difficulty/{EASY|MEDIUM|HARD}` - Filter by difficulty
- `POST /api/auth/signup` - Register new user
- `POST /api/auth/login` - Login user

---

## Protected Endpoints (Need JWT Token)
- `POST /api/chatbot/message` - Send message to chatbot
- `GET /api/chatbot/history` - Get chat history
- `POST /api/interview/start` - Start interview
- `GET /api/interview/history` - Get past interviews
- `GET /api/resume/history` - Get resume analysis history
- `GET /api/roadmap/today` - Get today's plan
- `GET /api/roadmap/generate` - Generate new plan
- `GET /api/roadmap/history` - Get past plans

---

## Admin Endpoints (Need ADMIN token)
- `POST /api/admin/problems` - Add coding problem
- `PUT /api/admin/problems/{id}` - Update problem
- `DELETE /api/admin/problems/{id}` - Delete problem
- `GET /api/admin/problems` - Get all problems (admin view)
- `GET /api/admin/students/progress` - Get all students' progress
- `GET /api/admin/students/{id}/progress` - Get student progress
